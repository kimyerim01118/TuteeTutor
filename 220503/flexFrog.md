# flex 속성들
### display : flex
flex를 사용하기 위해 container에서 줘야하는 속성

```css
.container{
    diplay:flex
}
```

### flex-dirction : 
**flex할 방향 지정**

```css
.container{
    diplay:flex
    flex-dirction:row / column
}
```

### justify-content : 
direction 방향을 기준으로 가로정렬
속성으로
* flex-start
* flex-end
* center
* space-between
* space-around

### align-item :
속성은 위와 같음
